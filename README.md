# babcocktech-week

This is a Next.js project designed for the Babcock Technology Week (BTW 2026) Hackathon & Ideathon platform.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
